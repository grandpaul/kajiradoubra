## 說明

本計畫為一可穿戴式設備. 透過手機藍芽遙控設備震動.

設備端為胸罩, 外加兩個小型震動馬達. 連結 Arduino.
Arduino 連接藍芽模組. 並用 motor driver 控制震動馬達.

手機為 Android 手機, 安裝我們做的 app, 透過藍芽來遙控 Arduino.
